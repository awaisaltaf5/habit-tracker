# Habit Tracker

A responsive habit tracker web app built with HTML, CSS, and JavaScript.

## Features

- Add habits
- Rename habits
- Delete habits
- Weekly tracking grid
- Streak calculation
- Week navigation
- LocalStorage persistence
- Responsive design

## Run Locally

1. Download the project
2. Open the folder
3. Open `index.html` in browser

OR use VS Code Live Server extension.

## Tech Stack

- HTML
- CSS
- JavaScript
