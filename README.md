# Habit Tracker

A responsive habit tracker web app built with HTML, CSS, and JavaScript.

## Live Demo

https://habit-tracker11.netlify.app/

## Features

- Add habits
- Rename habits
- Delete habits
- Weekly tracking grid
- Current streak calculation
- Week navigation
- LocalStorage persistence
- Responsive design
- Empty state handling

## Run Locally

### Option 1 — Open directly

Download the project and open:

index.html

in your browser.

### Option 2 — VS Code Live Server

1. Install the Live Server extension
2. Right-click index.html
3. Click "Open with Live Server"

## Tech Stack

- HTML
- CSS
- JavaScript
